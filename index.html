📱 새 기기로 이전하는 방법
1 현재 기기에서 전체 백업(.json) 다운로드
2 파일을 클라우드(카카오톡·이메일·메모앱)로 새 기기에 전송
3 새 기기에서 앱 열기 → 💾 버튼 → 가져오기
✓ 모든 대화·노트·분석이 완벽하게 복원됩니다

📚 사고력 OS 사용법

이 앱은 ChatGPT API를 활용하여 12단계 독서 사고력 분석을 수행합니다.

1
왼쪽 사이드바에 OpenAI API 키(sk-...)를 입력하고 저장하세요.
2
원하는 GPT 모델을 선택하세요. GPT-4o 권장.
3
책 제목, 문장, 문단, 또는 나의 생각을 입력하세요.
4
AI가 12단계 심층 분석을 제공합니다. 멘토의 질문에 답하며 사고력을 훈련합니다.
5
AI 응답 아래 [📒 노트 저장] 버튼을 누르면 분석 결과가 독서 노트에 저장됩니다.
6
상단 [📒 독서 노트] 탭에서 저장된 노트를 검색·편집·내보내기할 수 있습니다. Ctrl+S 로 빠른 저장.

⚠️ API 키는 브라우저 localStorage에만 저장되며 외부로 전송되지 않습니다.
